# Encompass Marketing — Voice and Copy Rules

16. PACT tier names are Foundation, Performance, Enterprise. Never "PACT 1/2/3" or "Standard" as a tier name. Display tier name prominently with a small "PACT" eyebrow above (unless the name is in a compact reference context like sticky headers, where just the tier name is used).

17. Ampersands (&) are reserved for section headers and nav labels only. Body copy, card titles, and descriptions spell out "and". Card titles like "Rate and Revenue Strategy Review" use "and," not "&".

18. En-dashes are permitted in numeric ranges only (e.g., "1-5 sites", "$80-250 per shop"). Prose sentences and titles use "to" or commas instead. Em-dashes remain prohibited everywhere.
